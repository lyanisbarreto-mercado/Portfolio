CSS Project - Art Fight Recreation

I was tasked with recreating a webpage on a site I frequent, using many CSS elements to create it. Art Fight is a website where users can “battle” each other through drawing characters and gaining points for their team. It is a fun event about gifting and appreciating ones creation, and I frequent the site to update information for the upcoming event.


You can look at the site in action here: https://lyanisbarreto-mercado.github.io/Art-Fight-Homepage-Recreation/
